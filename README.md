# Nano Banana Bot

![Nano Banana Bot Banner](https://placehold.co/600x200?text=Nano+Banana+Bot)

## Обзор
Nano Banana Bot — это Telegram-бот с интегрированным Mini App (Web App), который использует нейросеть **Nano Banana** от Google для генерации высококачественных изображений. Бот позволяет пользователям настраивать параметры генерации (соотношение сторон, разрешение), вводить промпт и прикреплять референсные изображения.

## Функционал
- **Интеграция Telegram Mini App**: Удобный веб-интерфейс внутри Telegram.
- **Продвинутая генерация изображений**: На базе модели Nano Banana.
- **Настраиваемые параметры**: Контроль разрешения, соотношения сторон и других параметров.
- **Мультимодальный ввод**: Поддержка текстовых промптов и референсных изображений.
