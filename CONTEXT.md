# Контекст Проекта

**Название проекта**: Nano Banana Bot
**Цель**: Создать Telegram Mini App для генерации изображений с использованием модели "Nano Banana" от Google.

## Технологический стек (Ожидает подтверждения)
- **Бот**: Python (aiogram) ИЛИ Node.js (grammy/telegraf)
- **Web App**: React / Vue / Vanilla JS
- **Бэкенд**: FastAPI / Express (если нужно проксирование API)
- **База данных**: SQLite / PostgreSQL (для настроек/истории)

## Ключевые точки интеграции
1. **Telegram Bot API**: Для взаимодействия в чате и запуска Web App.
2. **Telegram Web Apps Script**: Взаимодействие через `tg.initData`.
3. **Nano Banana API**: Провайдер генерации изображений.

## Текущее состояние
- Проект инициализирован.
- Документация переведена на русский язык.
- Ожидание уточнений от пользователя по конкретным деталям API и предпочтительному стеку.
